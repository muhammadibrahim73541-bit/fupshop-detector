# FupShop Detector v2.0

AI-powered phishing detection for Danish e-commerce webshops.

## Features
- 🔍 URL scanning for typosquatting & fake shops
- 🤖 LLM-powered reasoning (OpenRouter)
- 🔒 SSL/WHOIS/DNS verification
- 📊 W&B tracking & drift detection
- 🇩🇰 Danish CVR registry check

## Tech Stack
- XGBoost (94.51% accuracy)
- Flask + Responsive UI
- VirusTotal API
- Weights & Biases

## Usage
Enter any Danish webshop URL to get instant risk assessment.
